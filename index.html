个人简介

开源与个人基建

QemuRTOS

从零搭建的基于 QEMU 的类 Linux RTOS 开发环境。适配 RISC-V 32/64 virt 与 ARM Cortex-M/AArch64 virt。实现了内核 Bring-up、动态内存管理、UART 驱动，高度容器化并支持一键编译调试，是研究 GIC/PLIC 硬件中断路由的极佳实验平台。

C RTOS RISC-V/ARM QEMU

QemuEmbeddedLinux

为深度研习 Linux Foundation (LFD420) 高级课程打造的嵌入式 Linux 构建基座。整合 Buildroot、ARM ATF 与 U-boot，支持一键生成 rootfs 并在 QEMU 中无缝拉起。极大简化了内核模块 (Kernel Module) 开发、设备驱动测试及系统级调优的准入门槛。

Linux Kernel Buildroot U-boot Device Tree

EmbeddedDevContainer

彻底消除“在我的机器上能跑”痛点的嵌入式开发环境容器化基建。包含从零编写的多个高鲁棒性 Dockerfile，支持自动化编译 GNU Toolchain (GCC/GDB) 及快速挂载 ARM/RISC-V 交叉编译流水线。
